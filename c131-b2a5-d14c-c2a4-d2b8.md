# 성능 테스트란

### 성능테스트

시스템이 특정 상황에서 어느정도의 성능을 보이는지를 측정하는 것.

측정하는것은 용량 \(Capacity\) 

### 부하테스트

임계치까지 시스템에 부하를 단계적으로 증가시키는 테스트

장비가 가지는 한계치를 측정하고, 오류 \(메모리 릭 등\) 을 잡아내기 위한 테스트

### 스트레스테스

시스템에 강한 부하를 주어서 과부하 상태에서 시스템 상태를 측정하는 테스트
