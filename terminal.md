### asciinema

https://asciinema.org

쉽게 레코딩이 가능하다. 

로컬저장은 아니고 자체 서버로 업로드되는듯.

```bash
$ asciinema rec           // 녹화 시작
$ exit                    // 녹화 종료 (또는 Ctrl + D)
```
